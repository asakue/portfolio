Политика конфиденциальности

Последнее обновление: 11.03.2025

1. Основная информация

Сайт https://asakue.github.io/me/ является персональным портфолио и не осуществляет коммерческой деятельности. Эта политика объясняет принципы работы с информацией посетителей.

2. Какие данные собираются

  • Технические данные: GitHub Pages автоматически регистрирует базовую информацию о посещениях (IP-адрес, тип браузера, метки времени) согласно политике приватности GitHub.
  • Добровольные данные: На сайте отсутствуют формы сбора информации, cookies и системы аналитики. Любые данные, переданные через внешние контакты (email, соцсети), регулируются политиками соответствующих сервисов.

3. Использование информации

Данные используются исключительно для:

  • Обеспечения технической работоспособности сайта
  • Анализа и предотвращения сбоев (через логи GitHub Pages)

4. Безопасность данных

Сайт размещен на GitHub Pages — платформе, соответствующей стандартам безопасности разработчиков. Подробнее: GitHub Security.

5. Ваши права

Вы можете:

  • Запросить информацию о связанных с вами данных
  • Потребовать удаление таких данных (при наличии)

Для запросов используйте контактную информацию в конце документа.

6. Внешние ресурсы

Сайт не использует:

  • Сторонние аналитические системы
  • Рекламные скрипты
  • Встроенные плагины социальных сетей

Ссылки на внешние ресурсы открываются в новой вкладке и не передают данных о посетителях.

7. Изменения в политике

Все изменения публикуются на этой странице с обновлением даты в заголовке.
